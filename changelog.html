Changelog

All updates, improvements, and fixes.

The Big One

v4.0.0 — latest
March 2026

The largest update in Incorp's history. Rebuilt progression from the ground up, added 5 gambling games, a full company simulator, interactive bank heists, 15 image effects, real-time voice transcription, a smart AFK system, and a completely new anti-cheat layer — plus dozens of bug fixes.

XP & Rank System

  • 11-tier rank ladder: Novice → Apprentice → Adept → Expert → Master → Grandmaster → Legend → Mythic → Divine → Celestial → Transcendent
  • Each rank gives a permanent earnings multiplier — up to 1.75x at Transcendent
  • +1 Token reward on every rank-up
  • Global XP = server messages + commands used, both tracked with anti-spam cooldowns
  • Profile card shows an XP progress bar to your next rank threshold

Job Level System

  • 10-tier job progression tied to .work: Intern → Employee → Senior Employee → Team Lead → Manager → Director → VP → C-Suite → CEO → Industry Legend
  • Payout multiplier scales from 1.0x at Intern to 3.0x at Industry Legend
  • XP earned every work session; auto level-up with an in-chat announcement
  • .joblevel / .job — see your tier, XP progress, and next milestone

Rebirth System

  • Unlimited rebirth levels — reset wallet and bank to 0 for permanent stacking multipliers
  • Bonuses per level: +20% daily, +15% work, +10% beg, +5% rare fish, +10% slots, +20% XP
  • Cost scales 2.5x per rebirth starting at 100,000 coins
  • Requires typing 'confirm' in 30 seconds; 5-minute lock prevents pre-transfer exploits
  • Rebirth level shown as [RN] tag on the leaderboard

Upgrades — Expanded to 11

  • multiplier — +10% all earnings per level (max 20)
  • cooldown — -5% work/beg cooldown per level (max 5)
  • fishing — +15% fishing rewards per level (max 10)
  • gambling — +12% gambling winnings per level (max 5)
  • interest — +0.5% daily bank interest per level (max 5)
  • vault — +25% balance shielded from robbery per level (max 4)
  • xp — +20% XP gain per level (max 8)
  • company — unlocks company system + 10% income bonus (50M coins)
  • energy — -5% energy usage per level (max 10)

Gambling Overhaul

  • 500B coin win cap per 2-hour rolling window across all gambling commands
  • Cap checked upfront — blocked before the game runs, not after
  • Slots reweighted: Cherry 35%, Orange 25%, Lemon 20%, Diamond 12%, 7 6%, Jackpot 2%
  • Personal gambling toggle and bet limit cap in settings
  • Gambling window state saved to disk and survives restarts

Whale Pass

  • 6-tier marketplace upgrade that expands your 2-hour gambling win cap
  • Tier 1 — 1T (2 tokens) | Tier 2 — 1.5T (5) | Tier 3 — 2T (10)
  • Tier 4 — 2.5T (20) | Tier 5 — 3T (40) | Tier 6 — Unlimited (150)
  • Sequential — must own previous tier to unlock next
  • .gl / .gamble limit — check your current tier and cap

New Gambling Commands

  • .dice <amount> — roll 1–6; high wins 2x, low loses; 10s cooldown
  • .duel @user [amount] — PvP dice duel; 60s to accept; higher roll wins; 30s cooldown
  • .rune — rune-based gambling with 52% base win rate and escalating tiers
  • .diffuser — bomb diffuser; pick the right wire to build a multiplier, wrong wire loses all
  • .coinstreak — consecutive coinflip streak game; winning streak builds a multiplier

Rob System — Reworked

  • 70% base success chance with a 10-second suspense delay
  • Catch penalty: lose 30% of the attempted amount if caught
  • Vault upgrade reduces how much of a target's balance is stealable
  • rob_protect setting — toggle robbery immunity in settings
  • Simultaneous rob protection — can't rob someone already being robbed

Bank Interest

  • Fires automatically every hour; checked once per day per user
  • Rate: 0.5% per interest upgrade level, up to 2.5% at max
  • Requires the bank account upgrade to be active
  • DMs you the amount earned and your current rate each payout
  • Toggle with bank_interest setting

Fishing — Improvements

  • .fishquest [claim] — daily quests with a target fish and coin reward; progress tracked live
  • Fish locations — cast in different spots with unique catch pools per location
  • 55-rod progression; catch rate 1.0x–28.0x, rare chance 1.0x–9.0x, up to 18.5B coins
  • Fishing upgrade gives +15% rewards per level, stacking with rebirth

Autofish — Improvements

  • Stats tracked per session and all-time: caught, earned, rare, mythical
  • Auto-bait purchasing — .autofish bait <name> buys bait automatically when you run out
  • Autofish messages excluded from XP tracking — no passive XP farming
  • 2-hour daily limit persists across restarts

Company System

  • Full business simulator — requires Business License upgrade (50M coins)
  • .createcompany — found a Startup, Established, Enterprise, or Corporation
  • .upgradecompany — upgrade Office, Marketing, Technology, or Staff for income multipliers
  • .collectincome — collect daily income; Business License gives +10% bonus
  • .invest @user — invest in players' companies and share their profits proportionally
  • .hire / .unhire — manage employees; .createdepartment — add income streams
  • .companies — browse all active companies; .companystats — financial breakdown

Bank Heist System

  • .robbank — multi-stage heist; bank opens 4 windows daily at UTC 0–2, 6–8, 12–14, 18–20
  • Real-time progress bar; full 100% gives a 1.5x loot multiplier
  • Loot scales with your wallet — up to 3x base at high wealth
  • .bankstatus — check if bank is open and when the next window opens
  • .bankdebt / .paydebt — failed heists leave debt that must be paid before certain actions
  • .robstats — personal heist record; .quickrob — fast single-target rob

Voice Chat Tools

  • .vcscribe / .vctranscribe — real-time VC transcription with speaker names and timestamps
  • .vcsummary — AI-generated session summary; up to 8 concise bullet points

Games

  • .connect4 / .c4 — canvas board + GIF replay; vs bot (4 difficulties) or another user; bet support
  • .tictactoe / .ttt — canvas board; bot uses fork detection for stronger mid-game play
  • .wordle — classic 6-guess word game
  • .quiz — trivia with multiple choice answers
  • .wyr — Would You Rather; server votes on two random scenarios
  • .guessemoji — guess the concept from emoji combinations
  • .pictionary — collaborative drawing; full canvas tool suite on a 500×500 canvas

Image Effects

  • 15 effects: .pixelate, .blur, .mirror, .invert, .deepfry, .glitch, .sepia, .sharpen, .swirl, .vignette, .implode, .bandicam, .legoify, .speechbubble, .addtext
  • All effects accept a reply, attachment, or URL — consistent across every command
  • .effects — lists every available effect; .tools — full image/media/voice/TTS reference
  • .banner — view your or any user's Discord profile banner
  • Legoify rewritten with real LEGO brick textures — raised stud, shading, hard-light tinting
  • Glitch effect added — row-based horizontal pixel shift for chromatic-style distortion

TTS — Improvements

  • FIFO queue system — messages process in strict order, no overlapping audio
  • URL stripping — links replaced with just the domain name before reading
  • Voice state announcements — bot reads join, leave, mute, and deafen events
  • .ttsrate — adjustable speech speed; .ttsvolume — per-session volume; .ttsfun — voice effects
  • Special character voices: ghostface, chewbacca, c3po, stitch, stormtrooper, rocket, and more
  • ElevenLabs 3,000 character daily limit enforced per user; usage persists across restarts

AFK System

  • .afk [reason] — go AFK with an optional reason; anyone who mentions you gets notified
  • Clears automatically the moment you send any non-command message
  • Mention log on return — who pinged you, which channel, the time, and a message snippet
  • Every unique pinger gets an in-chat alert that you're back
  • 30s per-channel cooldown per user prevents notification spam
  • Spam protection — 5 AFK pings within 2 minutes auto-blacklists the sender for 2 hours

User Settings — Expanded

  • 16 toggleable settings enforced at the command level before any command runs
  • passive_income, bank_interest, rob_protect, gambling, pvp_challenges, bet_limit, dm_notifs, cd_remind, lvlup_msg, public_profile, leaderboard, analytics, romance, family_invites, active_theme, equipped_flag
  • Blocked commands show a clean denial with a link to incorp.live/settings

Anti-Cheat & Security

  • Suspicious activity monitoring across 18 commands with automatic flagging
  • Flags: single gain over 2M coins, total wealth over 50M, 1M+ in 15 seconds
  • Daily rotating command logs; suspicious events written to a separate log file
  • .auditlog — owner-only suspicious event viewer; .checkuser — full user data inspection
  • Bot account detection — auto temp-blacklist 30 days + owner DM notification
  • Command spam protection — 10 commands in 10 seconds triggers a 2-hour temp-blacklist
  • Hourly snapshots of 14 critical data files; last 24 retained (.snapshots / .restoreuser)

Website

  • Settings page — full UI for all 16 user preferences
  • Marketplace — Whale Pass card with tier display, next cost, and Maxed Out state
  • Marketplace — Gift system on every theme and AI voice card; gift by Discord ID with autocomplete
  • Profile settings tab — set your bio directly from the marketplace
  • 404 page — proper not-found handling

Bug Fixes

  • .server — channel counts now correct; was always 0 due to empty cache before filtering
  • .server — role names now shown sorted by position; was displaying count only
  • .profile — active pet now displays correctly; was reading from a non-existent file
  • AFK notifications no longer loop — bot's own reply re-triggered the handler indefinitely
  • .give / .deposit / .withdraw half — now correctly calculates 50%; was passing literal string 'half'
  • Autofish marketplace card missing closing div — Themes and Profile tabs were hidden for all users
  • Duplicate .trade and .work handlers removed — second unreachable case silently dropped
